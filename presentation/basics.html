CSS Keyframe Animations

What is a CSS Keyframe Animation?

CSS keyframes allow us to animate HTML elements without using JavaScript. This is accomplished by changing the styles from one specification to another. For example, we can change the background color of an element from black to white.

Templet

@keyframes animationname {
 keyframes-selector {css-styles;}
}

Example

@keyframes example {
 0% {background-color: #153e5c;}
 100% {background-color: #c1cdd5;}
}

In this example we are changing the background color from a darker blue (#153e5c) to a lighter blue (#c1cdd5). Click the box below to watch this happen.

Binding to an element

In order for your animations to work you need to bind the @keyframe to an element. This is done by adding the property name "animation-name" to the element's selector, and inserting that same name after your @keyframes selector as seen below.

Templet

element {
 animation-name: animationname;
}
@keyframes animationname {
 keyframes-selector {css-styles;}
}

Example

div {
 animation-name: example;
}
@keyframes example {
 keyframes-selector {css-styles;}
}

Beginning and Ending

You can change the CSS styles of an element as many times as you want in the same keyframe. To specify when a change will take place in the animation, use percentiles. At minimum you need to specify keyframes-selectors at 0% and 100%. These indicate where the animation starts (0%) and ends (100%). These can also be written as 'from' (equal to 0%) and 'to' (equal to 100%).

Templet

@keyframes animationname {
 0% {css-styles;}
 25% {css-styles;}
 50% {css-styles;}
 75% {css-styles;}
 100% {css-styles;}
}

Example

@keyframes animationname {
 0% {background-color: #153e5c;}
 25% {background-color: #ffffff;}
 50% {background-color: #698497;}
 75% {background-color: #000000;}
 100% {background-color: #153e5c;}
}

In this example we are changing the background color from a darker blue (at 0%) to white (at 25%) to a lighter blue (at 50%) to black (at 75%) and then back to the dark blue we started with (at 100%). Click the box below to watch this happen.

Multiple Styles

You can change as many CSS styles as you want in the same keyframe. To change multiple styles in the same keyframes-selector (or percent) add more styles within the curly brackets after the selector.

Templet

@keyframes animationname {
 0% {css-styles; css-styles; css-styles;}
 100% {css-styles; css-styles; css-styles;}
}

Example

@keyframes animationname {
 0% {background-color:#153e5c; height: 50px; width: 200px;}
 100% {background-color:#c1cdd5; height: 200px; width: 50px;}
}

In this example we are starting with a dark blue rectangle that is 200 pixels wide and 50 pixels tall, and ending with a light blue rectangle that is 50 pixels wide and 200 pixels tall. Click the box below to watch this happen.

Duration

The property name "animation-duration" is placed within the element's selector and defines how long the animation will take to complete. This property must be specified or no animation will begin because the default value for it is 0s, meaning zero seconds.

Templet

element {
 animation-name: animationname;
 animation-duration: #s;
}
@keyframes animationname {
 keyframes-selector {css-styles;}
}

Example

element {
 animation-name: example;
 animation-duration: 10s;
}
@keyframes example {
 0% {background-color:#153e5c; height: 50px; width: 200px;}
 100% {background-color:#c1cdd5; height: 200px; width: 50px;}
}

In this example we are changing the background color from a darker blue (at 0%) to white (at 25%) to a lighter blue (at 50%) to black (at 75%) and then back to the dark blue we started with (at 100%). The entire change will take ten seconds. Click the box below to watch this happen.
