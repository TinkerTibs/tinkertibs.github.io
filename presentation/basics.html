CSS Keyframe Animations

What is a CSS Keyframe Animation?

CSS keyframes allow us to animate HTML elements without using JavaScript. This is accomplished by changing the styles from one specification to another. For example, we can change the background color of an element from black to white.

@keyframes animationname {
keyframes-selector {css-styles;}
}

Animation!

Beginning and Ending

You can change the CSS styles of an element as many times as you want in the same keyframe. To specify when a change will take place in the animation, use percentiles. At minimum you need to specify keyframes-selectors at 0% and 100%. These indicate where the animation starts (0%) and ends (100%). These can also be written as 'from' (equal to 0%) and 'to' (equal to 100%).

@keyframes animationname {
0% {css-styles;}
25% {css-styles;}
50% {css-styles;}
75% {css-styles;}
100% {css-styles;}
}

Animation!

Multiple Styles

You can change as many CSS styles as you want in the same keyframe. To change multiple styles in the same keyframes-selector (or percent) add more styles within the curly brackets after the selector.

@keyframes animationname {
0% {css-styles; css-styles; css-styles;}
100% {css-styles; css-styles;} css-styles;}
}

Animation!
